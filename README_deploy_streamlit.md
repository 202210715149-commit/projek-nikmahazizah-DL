# Student Performance App (Streamlit)

Aplikasi ini menggunakan:
- Random Forest Regressor untuk prediksi **Current CGPA**
- K-Means untuk **clustering** mahasiswa

## File yang diperlukan

- `rf_pipeline.pkl`
- `kmeans_pipeline.pkl`
- `column_info.json`
- `streamlit_app.py`
- `requirements.txt`

## Cara menjalankan secara lokal

```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
```
